# Release Notes - v1.0.0

## Genel

İlk kararlı sürüm yayınlandı.
Bu sürüm restoran/kafe/bar işletmeleri için yönetilebilir dijital menü çözümünü içerir.

## Eklenen Özellikler

- Admin panelli dinamik menü yönetimi
- Kategori ve ürün CRUD
- Ürün görsel yükleme
- 3 menü görünümü (satır, kart, vitrin)
- 5 renk şeması
- 3 işletme teması (restoran, kafe, bar/pub)
- Otomatik indirim motoru
- İndirim badge stilleri
- Sürükle-bırak sıralama
- SEO yönetim ekranı
- QR kod üretme ve indirme
- Admin şifre değiştirme

## Teknik

- PHP + SQLite mimarisi
- MySQL gerektirmez
- Temiz route yapısı
- CSRF koruması
- Güvenli parola hash yapısı
